In diesem Projekt wird basierend auf Abschnitt 3 mithilfe von Schleifen eine Programm erzeugt welches ermöglicht mit der Maus
freihändig zu zeichnen.

Dieses Projekt deckt Freihand7 bis Freihand 9 ab.
